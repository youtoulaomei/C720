# JESD TX CIB 寄存器功能说明与排查手段

> 源文件: `jesd_tx_cib.v`  
> 模块名: `jesd_tx_cib`  
> 功能概述: JESD204B 发射通道的配置接口模块，提供 JESD IP 核的 AXI 配置桥、DDS 控制、链路状态监控

---

## 一、寄存器总览

### 基础寄存器组

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x00 | JESD_00 | R | 版本号 (VERSION) |
| 0x01 | JESD_01 | R | 编译日期 {YEAR, MONTH, DAY} |
| 0x02 | JESD_02 | R/W | JESD 复位控制: bit[0]=cfg_rst |
| 0x03 | JESD_03 | R/W | 测试寄存器 (写入取反存储) |
| 0x05 | JESD_05 | R/W | 通道选择: bit[1:0]=xn_sel |
| 0x06 | JESD_06 | R/W | 位置约束: bit[3:0]=PositionConstraint |

### AXI 桥寄存器组

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x10 | JESD_10 | R/W | AXI 地址: lb2axi_addr |
| 0x11 | JESD_11 | R/W | AXI 数据: lb2axi_data |
| 0x12 | JESD_12 | R/W | AXI 使能: bit[1]=读使能, bit[0]=写使能 |
| 0x13 | JESD_13 | R | AXI 读回数据: axi2lb_data |
| 0x14 | JESD_14 | R | AXI 忙状态: bit[1]=axi_rd_busy, bit[0]=axi_wr_busy |

### DDS 控制寄存器组

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x20 | JESD_20 | R/W | DDS 使能: bit[3:0]=dds_en |
| 0x21 | JESD_21 | R/W | DDS 配置触发: bit[0]=dds_config |
| 0x22 | JESD_22 | R/W | DDS 通道0配置数据 |
| 0x23 | JESD_23 | R/W | DDS 通道1配置数据 |
| 0x24 | JESD_24 | R/W | DDS 通道2配置数据 |
| 0x25 | JESD_25 | R/W | DDS 通道3配置数据 |
| 0x26 | JESD_26 | R/W | DDS 复位: bit[0]=dds_rst |

### 调试寄存器组

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x80 | JESD_80 | R | 历史告警: bit[2]=his_qpll_unlock, bit[1]=his_pll_unlock, bit[0]=his_tx_loss_sync |
| 0x90 | JESD_90 | R | SYSREF 计数: bit[9:0]=cur_sysref_cnt |
| 0x91 | JESD_91 | R | 全局频率计数: cur_glbl_freq_cnt |

## 二、寄存器详细说明

### 2.1 JESD_02 (0x02) - JESD 复位控制
- **类型**: 读写
- **位域**: bit[0] = `cfg_rst` (通过 assign cfg_rst = jesd_reset[0])
- **作用**: 对 JESD204B IP 核进行配置复位
- **排查用途**: 
  - JESD 链路异常时可写1进行复位
  - 复位后需重新配置 JESD IP 核参数

### 2.2 JESD_05 (0x05) - 通道选择
- **类型**: 读写
- **位域**: bit[1:0] = `xn_sel`
- **作用**: 选择 JESD 通道配置模式（x1/x2/x4）
- **排查用途**: 确认通道模式与硬件设计匹配

### 2.3 JESD_06 (0x06) - 位置约束
- **类型**: 读写
- **位域**: bit[3:0] = `PositionConstraint`
- **作用**: JESD 通道位置约束配置

### 2.4 AXI 桥寄存器 (0x10~0x14)
- **作用**: 提供从本地总线到 JESD IP 核 AXI 总线的配置桥
- **写操作流程**:
  1. 写 0x10 设置 AXI 地址
  2. 写 0x11 设置写数据
  3. 写 0x12 = 0x01 触发写操作
  4. 轮询 0x14 bit[0] 直到为0（写完成）
- **读操作流程**:
  1. 写 0x10 设置 AXI 地址
  2. 写 0x12 = 0x02 触发读操作
  3. 轮询 0x14 bit[1] 直到为0（读完成）
  4. 读 0x13 获取读回数据

### 2.5 DDS 控制寄存器 (0x20~0x26)
- **dds_en (0x20 bit[3:0])**: 各通道 DDS 使能开关
- **dds_config (0x21 bit[0])**: 触发 DDS 参数更新
- **dds_config_x_data (0x22~0x25)**: 各通道 DDS 频率/相位配置字
- **dds_rst (0x26 bit[0])**: DDS 模块复位

### 2.6 JESD_80 (0x80) - 历史告警
- **类型**: 只读（读清）
- **位域**:
  - bit[2]: `his_qpll_unlock` — QPLL 失锁历史
  - bit[1]: `his_pll_unlock` — PLL 失锁历史
  - bit[0]: `his_tx_loss_sync` — 发送失步历史
- **清除方式**: 读取本寄存器时自动清除
- **排查用途**: 
  - **QPLL 失锁**: 检查参考时钟是否稳定、GT 通道配置是否正确
  - **PLL 失锁**: 检查时钟源和分频配置
  - **TX 失步**: 检查 SYSREF 信号、JESD 参数配置

### 2.7 JESD_90 (0x90) - SYSREF 计数
- **类型**: 只读
- **位域**: bit[9:0] = `cur_sysref_cnt`
- **作用**: 累计 SYSREF 信号边沿计数
- **排查用途**: 
  - 确认 SYSREF 信号是否正常到达 FPGA
  - 若计数不变，说明 SYSREF 信号缺失

### 2.8 JESD_91 (0x91) - 全局频率计数
- **类型**: 只读
- **位域**: [31:0] = `cur_glbl_freq_cnt`
- **作用**: GT 通道参考时钟频率测量
- **排查用途**: 
  - 验证 GT 参考时钟频率是否与设计一致
  - 该值应在已知参考频率的合理范围内

## 三、问题排查手册

### 3.1 JESD 链路无法建立
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 确认版本 | 读 0x00, 0x01 | 版本日期正确 |
| 2. 检查告警 | 读 0x80 | 若非0则有链路问题 |
| 3. 检查参考时钟 | 读 0x91 | 频率值合理 |
| 4. 检查 SYSREF | 读 0x90 | 计数在增加 |
| 5. 复位 JESD | 写 0x02=0x01 | 重新初始化 |

### 3.2 PLL/QPLL 失锁
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 读告警 | 读 0x80 | bit[2:1]指示失锁类型 |
| 2. 检查参考时钟 | 硬件测量 | 频率稳定、抖动在范围内 |
| 3. 检查供电 | 测量 GT 电源 | MGTAVCC=1.0V, MGTAVTT=1.2V |
| 4. 复位重试 | 写 0x02=0x01 → 0x00 | PLL 重新锁定 |

### 3.3 DDS 输出异常
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 检查 DDS 使能 | 读 0x20 | 对应通道 bit 为1 |
| 2. 检查配置字 | 读 0x22~0x25 | 频率字正确 |
| 3. 触发配置更新 | 写 0x21=0x01 | DDS 参数生效 |
| 4. DDS 复位 | 写 0x26=0x01 → 0x00 | DDS 重启 |

### 3.4 AXI 配置桥卡死
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 读忙状态 | 读 0x14 | 若长时间非0则卡死 |
| 2. 复位 JESD | 写 0x02=0x01 | AXI 总线恢复 |
| 3. 检查地址 | 确认 0x10 地址有效 | 地址在 IP 核范围内 |
