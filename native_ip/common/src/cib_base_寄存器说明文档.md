# CIB Base (公共基础) 寄存器功能说明与排查手段

> 源文件: `cib_base.v`  
> 模块名: `cib_base`  
> 功能概述: 系统公共基础寄存器模块，提供设备标识、编译信息、FPGA DNA、温度/电压监控、中断管理、RTC 时间读取等系统级功能

---

## 一、寄存器总览

### 设备标识组

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x00 | BASE_00 | R | 设备 ID (VERSION) |
| 0x01 | BASE_01 | R | 编译日期 {YEAR, MONTH, DAY} |
| 0x02 | BASE_02 | R/W | 测试寄存器，复位值 0xDEAD_BEEF |
| 0x03 | BASE_03 | R | FPGA DNA 低32位 (dna_value[31:0]) |
| 0x04 | BASE_04 | R | FPGA DNA 高25位 (dna_value[56:32]) |

### 系统监控组

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x06 | BASE_06 | R | 温度读数: temperature[15:0] |
| 0x07 | BASE_07 | R | 电压 VCCINT: vccint[15:0] |
| 0x08 | BASE_08 | R | 电压 VCCAUX: vccaux[15:0] |
| 0x09 | BASE_09 | R/W | 温度告警使能: temp_alm_en_low, temp_alm_en_high |
| 0x0A | BASE_0A | R/W | 温度告警阈值: temp_low_limit, temp_high_limit |
| 0x0B | BASE_0B | R | 温度告警状态(读清): his_temp_low, his_temp_high |

### 中断/加载管理

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x0E | BASE_0E | R/W | 中断优先级: int_level (3级) |
| 0x0F | BASE_0F | R/W | 加载控制: rd_icap_en, wr_icap_en, icap_addr |
| 0x10 | BASE_10 | R | 上位机时间 年/月/日 |
| 0x11 | BASE_11 | R | 上位机时间 时/分/秒 |
| 0x12 | BASE_12 | R | 上位机时间 毫秒/微秒 |
| 0x13 | BASE_13 | R | 上位机时间 纳秒 |

### 中断寄存器 (INT_0~INT_7, 地址 0x70~0x7F)

| 地址 | 宏定义 | 读写 | 功能简述 |
|------|--------|------|----------|
| 0x70 | INT_0 | R/W | 中断0 使能/状态 |
| 0x71 | INT_0 | R | 中断0 屏蔽 |
| 0x72 | INT_1 | R/W | 中断1 使能/状态 |
| 0x73 | INT_1 | R | 中断1 屏蔽 |
| ... | ... | ... | 最多支持8级中断 |

## 二、关键寄存器详细说明

### 2.1 BASE_00 (0x00) - 设备标识
- **类型**: 只读
- **位域**: [31:0] = VERSION 参数
- **作用**: 唯一标识设备类型和版本
- **排查核心**: 系统启动后第一个应读的寄存器

### 2.2 BASE_02 (0x02) - 测试寄存器
- **类型**: 读写
- **复位值**: 0xDEAD_BEEF
- **作用**: CPU 总线完整性测试
- **排查方式**: 写入任意值后读回验证，是最基础的总线测试手段

### 2.3 BASE_03/04 (0x03~0x04) - FPGA DNA
- **类型**: 只读
- **作用**: 读取 FPGA 芯片内部唯一 DNA ID（57bit）
- **排查用途**: 
  - 设备唯一标识
  - 授权绑定验证
  - 当前使用 shift_dna 模块自动读取

### 2.4 BASE_06~08 (0x06~0x08) - 温度/电压监控
- **数据格式**: XADC 原始数据，16bit
- **温度转换**: T(°C) = (temperature × 503.975 / 65536) - 273.15
- **电压转换**: V = ADC_value × 3.0 / 65536
- **排查用途**: 
  - 系统过热保护
  - 供电异常检测

### 2.5 BASE_09/0A/0B - 温度告警系统
- **BASE_09 (0x09)**: 告警使能
  - bit[16]: `temp_alm_en_low` — 低温告警使能
  - bit[0]: `temp_alm_en_high` — 高温告警使能
- **BASE_0A (0x0A)**: 告警阈值
  - bit[31:16]: `temp_low_limit` — 低温阈值（XADC 原始值）
  - bit[15:0]: `temp_high_limit` — 高温阈值
  - 复位值: 0x4E82_D555 → 低温约-40°C，高温约85°C
- **BASE_0B (0x0B)**: 告警状态（读清）
  - bit[1]: `his_temp_low` — 低温告警历史
  - bit[0]: `his_temp_high` — 高温告警历史

### 2.6 BASE_0E (0x0E) - 中断优先级
- **类型**: 读写
- **位域**: bit[2:0] = `int_level` (0~7)
- **作用**: 设置中断优先级等级
- **排查用途**: 确认中断优先级配置正确

### 2.7 BASE_0F (0x0F) - ICAP 加载控制
- **类型**: 读写
- **位域**:
  - bit[27]: `rd_icap_en` — ICAP 读使能
  - bit[24]: `wr_icap_en` — ICAP 写使能
  - bit[23:0]: `icap_addr` — 多重启动地址
- **作用**: FPGA 多重启动/热重配置控制
- **注意**: 此寄存器操作可能导致 FPGA 重新加载，需谨慎

### 2.8 BASE_10~13 (0x10~0x13) - 上位机时间
- **类型**: 只读（CPU 写入，FPGA 读取）
- **位域**: 年/月/日/时/分/秒/毫秒/微秒/纳秒
- **作用**: 上位机向 FPGA 传递时间信息

### 2.9 中断寄存器组 (0x70~0x7F)
- **结构**: 每两个地址一组（使能+屏蔽）
- **使能寄存器**: 可读写，控制各中断源使能
- **屏蔽寄存器**: 只读，指示当前中断状态
- **排查用途**: 中断系统调试

## 三、问题排查手册

### 3.1 系统启动自检流程
```
1. 读 0x00 验证设备 ID            → 确认 FPGA 逻辑正确加载
2. 读 0x01 验证编译日期            → 确认版本正确
3. 写回环测试 0x02                 → 验证 CPU 总线通路
4. 读 0x03~0x04 获取 DNA          → 设备唯一标识
5. 读 0x06 获取温度                → 确认芯片温度正常
6. 读 0x07~0x08 获取供电电压       → 确认供电正常
```

### 3.2 CPU 总线不通
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 读默认值 | 读 0x02 | 应返回 0xDEAD_BEEF |
| 2. 写回环 | 写 0x02=0x12345678 后读 | 读回 0x12345678 |
| 3. 反转测试 | 写 0x02=0xAAAAAAAA 后读 | 数据位无粘连 |
| 4. 检查硬件 | 确认 CPU 总线连接 | 信号线完好 |

### 3.3 FPGA 温度过高
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 读当前温度 | 读 0x06 | 转换后 < 85°C |
| 2. 读告警标志 | 读 0x0B | 确认是否触发过温 |
| 3. 检查散热 | 确认风扇/散热片 | 散热正常 |
| 4. 降低负载 | 关闭不必要的通道 | 功耗降低 |
| 5. 调整阈值 | 写 0x0A | 根据需求调整 |

### 3.4 供电电压异常
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 读 VCCINT | 读 0x07 → V = val×3/65536 | 应约 1.0V |
| 2. 读 VCCAUX | 读 0x08 → V = val×3/65536 | 应约 1.8V |
| 3. 测量实际电压 | 万用表测量 | 与读回值一致 |
| 4. 检查电源模块 | DC-DC/LDO 输出 | 电压稳定 |

### 3.5 FPGA DNA 读取失败
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 读 DNA | 读 0x03, 0x04 | 非全0或全F |
| 2. 检查 shift_dna | 确认模块例化 | DNA 读取逻辑正确 |
| 3. 等待完成 | 上电后延迟读取 | DNA 序列读取需要时间 |

### 3.6 中断系统排查
| 排查步骤 | 操作 | 预期结果 |
|----------|------|----------|
| 1. 检查中断级别 | 读 0x0E | int_level 配置正确 |
| 2. 检查使能 | 读 0x70/72/74... | 中断源已使能 |
| 3. 检查屏蔽 | 读 0x71/73/75... | 中断未被屏蔽 |
| 4. 清除中断 | 寫使能寄存器 | 清除悬挂中断 |
