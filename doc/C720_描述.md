# BHD-C720 宽带信号采集与回放系统

---

## 版本一：详细版（适合项目经历栏）

### 项目名称
BHD-C720 宽带信号采集与回放系统 — FPGA逻辑设计

### 项目描述
基于 Xilinx Kintex-7 FPGA 的宽带信号采集与回放平台，实现双通道高速ADC采集、数字信号处理（DDC/DUC/BPF）、PCIe Gen3 x8高速数据传输、DDR3数据缓存调度、JESD204B高速DAC回放，以及北斗/GJB-B码双时间源纳秒级同步授时功能。系统应用于电子信息领域的信号侦收、记录与回放场景。

### 工作内容
- 负责FPGA顶层架构设计与模块集成，完成15+功能模块的例化、时钟规划和数据通路搭建
- 设计并实现多种位宽转换模块（16→64、32→64、512→32等），解决多时钟域、多位宽数据通路的适配问题
- 基于自研异步FIFO（格雷码指针同步方案）实现跨时钟域数据传输，覆盖5个独立时钟域
- 实现CPU总线地址分配框架（32用户×256字节地址空间），统一管理PCIe至各子模块的寄存器读写访问
- 完成7路SPI控制器的集成与调试，配置ADC（AD9467）、DAC（AD9172）、PLL（AD9516/HMC7044）及变频模块（ZD20E/ZC18）
- 设计北斗/B码双时间源选择逻辑与PPS健康检测机制，支持外部PPS/内部定时器自动回退
- 完成NOR Flash在线升级通路（PCIe→DDR3→512-to-16位宽转换→MEFC Flash写入）
- 使用Vivado ILA进行关键信号抓取与链路调试，设计多级告警监测寄存器辅助系统状态诊断

## 面试自我介绍参考（30秒版）

> C720项目是一个宽带信号采集与回放系统，我基于Kintex-7 FPGA完成了整个系统的逻辑设计。采集侧通过LVDS接收双通道ADC数据，经BPF滤波和DDC下变频后，通过数据复用器打包存入DDR3，再经PCIe Gen3 x8上传主机；回放侧从PCIe接收数据经DUC上变频后通过JESD204B发送到DAC。系统涉及5个时钟域的跨域处理、自研异步FIFO设计、7路SPI外设配置，以及北斗和B码双时间源纳秒级同步。
